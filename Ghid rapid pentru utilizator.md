# Ghid rapid pentru utilizator

## Login admin

1. Deschide: https://domeniul-tău.ro/admin
2. Loghează-te cu email + parolă.

## Editează conținutul

1. Alege secțiunea dorită.
2. Modifică textul sau imaginea.
3. Apasă Save.

## Noutăți și Comunitate

1. Editezi postările din posts.
2. Publici doar comentariile aprobate din approvedComments.
3. Pentru fiecare comentariu setezi trustLevel:
   - nou
   - verificat
   - confirmat

## Publică modificările

1. Apasă Publish.
2. Așteaptă 1-2 minute (deploy Netlify).
3. Verifică site-ul live.

## Regulă anti-greșeală

1. Modifici puțin.
2. Save.
3. Publish.
4. Verifici live.

## Dacă nu merge /admin

1. Netlify Identity = Enabled
2. Netlify Git Gateway = Enabled
3. Invitația pe email este acceptată
4. Reîncarcă pagina și încearcă din nou
