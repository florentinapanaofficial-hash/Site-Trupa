# Ghid printabil A4 - Administrare site

Acest ghid este varianta simplă pentru print.

## 1) Login în admin

1. Deschide adresa: https://domeniul-tău.ro/admin
2. Introdu emailul și parola.
3. Apasă Log in.

## 2) Editează conținutul

1. Alege secțiunea dorită.
2. Modifică textul sau imaginea.
3. Apasă Save.

## 3) Noutăți și Comunitate

1. Postările sunt în posts.
2. Comentariile publice sunt în approvedComments.
3. Publici doar comentariile aprobate de tine.
4. Setezi nivelul de încredere pentru fiecare:
   - nou
   - verificat
   - confirmat

## 4) Publică modificările

1. Apasă Publish.
2. Așteaptă 1-2 minute.
3. Verifică site-ul live.

## 5) Verificare înainte de lansare

1. Domeniul real este setat la SITE_URL.
2. Sitemap-ul din robots.txt are domeniul real.
3. Datele SEO sunt completate.

## 6) Dacă nu merge /admin

1. Verifică în Netlify: Identity = Enabled.
2. Verifică în Netlify: Git Gateway = Enabled.
3. Confirmă invitația pe email.
4. Reîncarcă pagina și încearcă login.

## 7) Regulă simplă de lucru

1. Modifici puțin.
2. Save.
3. Publish.
4. Verifici live.

Astfel ai control clar și eviți greșeli.
